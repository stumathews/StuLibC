Detailed Description

Timing library prototypes.

Author
Stuart Mathews
Date
19 July 2013

This header contains the function and type declarations for dealing with timing functionality in the library. This includes measuring time such as timing how long something takes or implementing a stopwatch or count down timer etc..

See also
http://devel.stuartmathews.com/stulibc
