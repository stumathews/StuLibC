◆ STR_Join()

LIBRARY_API char* STR_Join ( const char *  s1,
const char *  s2 
)

Joins two strings together.

Creates a new string by joining string1 and string2 together. The new string will be allocated by the function.

Note: This string is allocated by the function and needs to be deallocated by the caller. Internally the following c runtime function are used 

 strcpy(joined, string1);
 strcat(joined, string2);

Example of usage:

char expected[] = "one two";
char* append_result = (char*) STR_Join( "one ", "two");
Parameters
string1The first string
string2The second string which will be appended to string1.
Returns
A new string allocated in memory with string2 appended to string1 *
See also
http://www.stuartmathews.com
Note
If the string cannot be allocated dynamically, the program exists
Warning
This allocates memory from within and doesn't free it. This is probably not good use in a TSR or long running program
