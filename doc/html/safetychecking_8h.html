Detailed Description

Safety checking library prototypes.

Author
Stuart Mathews
Date
19 July 2013

This header contains the function and type declarations for dealing with safety functionality in the library. This includes ensuring bounds are not overflowed, and that where restrictions are enforced, these are upheld.

See also
http://devel.stuartmathews.com/stulibc
