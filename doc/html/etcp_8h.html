Detailed Description

Networking library routines.

Author
Stuart Mathews
Date
22 March 2015

This header contains the function and type declarations for dealing with low level networking functionality in the library. This includes reading and writing to sockets and tcp and udp client and server routines

See also
http://devel.stuartmathews.com/stulibc
