Detailed Description

File management library prototypes.

Author
Stuart Mathews
Date
19 July 2013

This header contains the function and type declarations for dealing with file management functionality in the library. This includes writing text to files, binary data to files and records etc.

See also
http://devel.stuartmathews.com/stulibc
